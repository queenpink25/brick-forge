
BRICK FORGE GAME

A classic brick breaker game built with HTML5 Canvas, CSS, and JavaScript. Break all the bricks to win!

https://queenpink25.github.io/brick-forge/

 Features
 5 Different Levels - Each with unique brick patterns

 Score System - Earn points for breaking bricks

❤️ Lives System - Start with 3 lives, game over when they reach 0

 Responsive Design - Works on desktop and mobile

 Multiple Controls - Keyboard and touch controls

 Clean UI - Modern design with smooth animations


How to Play
Objective
Break all the bricks using the ball without letting it fall below the paddle.


Controls
Keyboard:

← → Arrow keys or A D keys: Move paddle

SPACEBAR: Launch ball / Start game

P key: Pause/Resume game

R key: Restart game

Mobile/Touch:

Use the left/right arrow buttons

Tap LAUNCH button to start


Scoring
Normal brick: 10 points

Strong brick (2 hits): 20 points


Special brick: 30 points
